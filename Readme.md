
# metalsmith-textile

  A Metalsmith plugin to convert textile files.

## Installation

    $ npm install metalsmith-textile

## CLI Usage

  Install via npm and then add the `metalsmith-textile` key to your `metalsmith.json` plugins, like so:

```json
{
  "plugins": {
    "metalsmith-textile": {}
  }
}
```
